# ☕︎ Primeiro Ano de CCOMP

Repositório contendo materiais, exercícios e projetos das disciplinas do **primeiro ano** do curso de **Ciência da Computação** na UNESPAR.

## ☕︎ Sobre o Repositório

Este repositório reúne o conteúdo estudado ao longo do primeiro ano, organizado por disciplina:

- *Algoritmos e Técnicas de Programação*

## ☕︎ Objetivo

Centralizar os códigos, exercícios e anotações produzidos durante o primeiro ano, servindo como material de estudo, revisão e portfólio acadêmico.

## ☕︎ Tecnologias e Linguagens Utilizadas

- C;
- Code::Blocks.

## ☕︎ Como Usar

## Como clonar este repositório

```bash
git clone https://github.com/MariaCampanaP/Primeiro-Ano-de-CCOMP.git
```

Isso vai baixar todas as disciplinas do 1° ano.
Se quiser baixar apenas a pasta de uma disciplina específica, veja as instruções de sparse-checkout no README de cada pasta.
